# Thermal-Mass-Diff-3D
### 3次元空間における熱・物質の移流拡散数値シミュレーション

## 概要
気相成長中の微細な金属原子の挙動を探るため、
熱と物質（濃度）の拡散を3次元空間でシミュレートするプログラムです。

設定可能な項目として、
・各軸のスケール、解像度
・各種物性値
を用意

## 技術的特徴
- **手法**: 有限差分法を用いた偏微分方程式の数値解法。
- **機能**: 時間発展に伴う空間内の熱分布・濃度分布の遷移を可視化。
- **応用先**: 電子機器の筐体設計における放熱シミュレーションや物理モデリングの習作として開発。

## 使用言語
- Python (NumPy, os)
